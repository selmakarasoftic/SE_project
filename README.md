# AutoVerse

AutoVerse is a web application for car enthusiasts. The system allows users to browse cars, read and create reviews, view gallery content, and manage platform data through backend services.

## Project Structure

The project is organized into frontend and backend parts.

- `frontend/` - contains HTML, CSS, JavaScript files and UI pages
- `backend/` - contains PHP backend logic, DAO classes, services, middleware, routes and configuration
- `backend/tests/` - contains backend unit tests

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
